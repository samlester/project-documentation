6.1 Methodology

For the testing of the application I decided to use a piece of user testing software called Silverback which runs in the background and records the test computer's screen as well as input from webcam and microphone.

A video can then be exported which shows the application and the participant's reactions alongside each other.

6.2 Participants and setup

I tested Scoop individually with with 6 members of the Ten4 Design team. Each participant was given an example username and password to log in to the system and an example site to crawl (Made in Brunel). To ensure that analytics data could be retrieved the relevant Google account was logged in prior to the start of the testing.

Before each test the participant was briefed on the purpose of the system and encouraged to give feedback on the application interface and functionality as they went along. Below I will outline the key points that arose from the testing for each participant.
